## Install
git clone http://github.com/lupin/tubecrawler

## Change Directory
cd tubecrawler

## ffmpeg install
Depending on your environment, you may need to install ffmpeg

## Dependency
pip install -r requirements.txt

uv pip install -r requirements.txt
